# PwnEngine
PwnEngine is a free, browser-based cybersecurity toolkit for developers, researchers, and enthusiasts. It allows you to quickly encode, decode, hash, parse, and analyze data—all safely in your browser. Perfect for learning, testing, or experimenting with data workflows, PwnEngine is lightweight, modular, and privacy-friendly, making it ideal for educational and authorized security research.

# PwnEngine Features 🛠️
>🅱️ Base64 Encoder/Decoder – Encode or decode text instantly.

>🌐 URL Encoder/Decoder – Safely format URLs for web use.

>🔒 Hash Generators – MD5, SHA1, SHA256 hashing.

>🧩 JSON Formatter & Parser – Clean, readable JSON output.

>🔑 Password & Token Generator – Create strong credentials safely.
